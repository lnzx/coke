# headscale web ui

使用web界面操作headscale,

### 支持功能:

- 安装headscale
- 自定义节点IP
- 管理节点
- 管理namespacce
- 

## api

### login

`POST /api/login`

`POST /api/logout`

### node

`GET /api/nodes`

`POST /api/nodes`

`PUT /api/nodes`

`DELETE /api/nodes`

### user namespace

`GET /api/ns`

`POST /api/ns`

`PUT /api/ns`

`DELETE /api/ns`
